GitHub 사이트에서 폴더를 만드는 방법은 다음과 같습니다.

해당 저장소(repository)로 이동하여 상단의 "Add file" 버튼을 클릭하고 "Create new file"을 선택합니다.
파일 이름 입력란에 폴더 이름과 슬래시(/)를 함께 입력합니다. 예를 들어 new-folder/라고 입력하면 new-folder라는 폴더가 생성됩니다.
폴더 안에 파일이 반드시 하나 있어야 폴더가 생성되므로 파일 이름도 입력해야 합니다. 예를 들어 new-folder/readme.md와 같이 입력합니다.
내용(빈 파일도 가능)을 입력하고 아래쪽에서 "Commit new file" 버튼을 클릭하여 커밋을 완료하면 폴더가 생성됩니다.
빈 폴더는 만들 수 없으며 반드시 내부에 파일이 있어야 폴더가 만들어집니다. 이 방법은 GitHub 웹사이트에서 직접 폴더를 만들 때 가장 간단한 절차입니다.
